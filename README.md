## :computer: Projeto

Aplicação web para pesquisa de repositórios utilizando a [API do GitHub](https://docs.github.com/en/rest/users?apiVersion=2022-11-28).


## ✔️ Recursos Utilizados

- `HTML`
- `CSS`
- `TypeScript`
- `React`
- `React Router Dom`
- `React Toastify`
- `React Router Dom`
- `Animate.css`
